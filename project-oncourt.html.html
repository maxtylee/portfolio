OnCourt Athletic Training Tracker

The OnCourt project is a wearable wrist-tracker designed to analyze forehand swings in tennis. It uses an MPU6050 sensor and ESP32 microcontroller with an OLED screen to record and display movement data. The design emphasizes compactness, real-time feedback, and distinguishable swing pattern detection between beginner and experienced players.


I prototyped and tested the device using PLA-printed enclosures and adjustable wristbands. Through recorded data, the system highlights inconsistent wrist motion in amateur swings compared to smoother curves in experienced athletes — enabling real-time coaching and self-correction.


Skills: Embedded Systems · Mechanical Design · Prototyping · Motion Analysis
