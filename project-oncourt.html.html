A lightweight wearable for real-time wrist motion tracking and swing correction in tennis.

Project Overview

OnCourt is a wrist-mounted wearable device built to analyze tennis forehand swings. Designed with the MPU6050 IMU and ESP32 microcontroller, it tracks rotational motion in real-time and displays live feedback on an OLED screen.

The enclosure was custom-modeled in CAD and 3D-printed using PLA, paired with a quick-release wristband. Data collected from beginner and experienced players highlighted variations in wrist stability and rotational smoothness, enabling players to improve form through quantitative feedback.
